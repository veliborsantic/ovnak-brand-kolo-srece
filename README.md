# Kolo Sreće - Brandovi

Next.js aplikacija sa kolo sreće komponentom koja prikazuje logotipe brandova.

## Pokretanje

Prvo instalirajte dependencies:

```bash
npm install
```

Zatim pokrenite development server:

```bash
npm run dev
```

Otvori [http://localhost:3000](http://localhost:3000) u browseru.

## Funkcionalnosti

- Kolo sreće sa logotipima brandova
- Animacija rotacije
- Prikaz pobjednika nakon završetka
- Responsive dizajn

## Zamjena placeholder slika

Zamijenite placeholder URL-ove u `app/page.tsx` sa stvarnim logotipima. Možete koristiti:
- Lokalne slike u `public/` folderu
- URL-ove do slika na internetu
- Next.js Image komponentu za optimizaciju
